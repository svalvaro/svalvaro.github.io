Proteomics Analyser

Web application for proteomics data analysis..

Web app for the quantitative analysis of proteomics data. Heatmaps, Volcano plot, enrichment, pathway analysis, etc. Everything that you need!

  • Created with: R, and some bits of JavaScript and CSS
  • Category: Web Application

ProteoViewer

Web application for protein visualization..

Web app for visualizing the protein topology, peptide intensity and Post-translational Modifications. It uses the Protter API.

  • Created with: R
  • Category: Web Application

MaxQuant Metrics

R package for proteomics quality control

The package MQmetrics provides a workflow to analyze the quality and reproducibility of your proteomics mass spectrometry analysis from MaxQuant. Input data are extracted from several MaxQuant output tables and produces a pdf report. It includes several visualization tools to check numerous parameters regarding the quality of the runs.

  • Created with: R
  • Category: R package

drawCell

The easiest way to generate Cell pictures

drawCell is an R package that allows the generation of any type of cell type with coloured organelles. It uses the SwissBioPics API.

  • Created with: R and JavaScript
  • Category: R package & (soon) Shiny app

draw and Predict

Shiny will predict whatever you write!

A small project of a Shiny app that uses a trained neural network to predict the letters that you write on Shiny.

  • Created with: R & Python
  • Category: Web application, Machine Learning
